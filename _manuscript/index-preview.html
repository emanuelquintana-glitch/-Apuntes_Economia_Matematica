1 Prefacio

Bienvenido a este manuscrito sobre Economía Matemática. Este trabajo surge de la necesidad de integrar rigurosamente las herramientas matemáticas con la teoría económica, proporcionando un puente entre el análisis formal y las aplicaciones prácticas en economía.

1.1 Objetivo del Texto

Este texto está diseñado para:

  • Proporcionar fundamentos sólidos en métodos matemáticos aplicados a economía
  • Desarrollar habilidades para modelar problemas económicos formalmente
  • Presentar técnicas de análisis estático y dinámico
  • Ofrecer aplicaciones prácticas en microeconomía y macroeconomía

1.2 Estructura del Manuscrito

El contenido está organizado en cinco partes principales:

1.2.1 Parte I: Introducción

Fundamentos conceptuales y metodológicos de la economía matemática.

1.2.2 Parte II: Análisis Estático

Modelos de equilibrio y álgebra de matrices aplicada a economía.

1.2.3 Parte III: Análisis Estático Comparativo

Técnicas de diferenciación y análisis de sensibilidad en modelos económicos.

1.2.4 Parte IV: Problemas de Optimización

Métodos de optimización con y sin restricciones aplicados a teoría económica.

1.2.5 Parte V: Análisis Dinámico

Sistemas dinámicos, ecuaciones diferenciales y teoría de control óptimo.

1.3 Características Pedagógicas

  • Ejemplos económicos en cada concepto matemático
  • Aplicaciones prácticas con Python/R
  • Ejercicios resueltos paso a paso
  • Conexiones con teoría económica moderna

1.4 Prerrequisitos

El texto asume familiaridad con: - Cálculo diferencial e integral - Álgebra lineal básica - Conceptos fundamentales de microeconomía

# Ejemplo introductorio: Función de utilidad Cobb-Douglas
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# Parámetros
alpha, beta = 0.6, 0.4
x = np.linspace(0.1, 10, 50)
y = np.linspace(0.1, 10, 50)
X, Y = np.meshgrid(x, y)

# Función de utilidad Cobb-Douglas
U = (X**alpha) * (Y**beta)

# Gráfico 3D
fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, U, cmap='viridis', alpha=0.8)
ax.set_xlabel('Bien X')
ax.set_ylabel('Bien Y')
ax.set_zlabel('Utilidad U(x,y)')
ax.set_title('Función de Utilidad Cobb-Douglas\n$U(x,y) = x^{0.6} y^{0.4}$')
plt.tight_layout()
plt.show()
