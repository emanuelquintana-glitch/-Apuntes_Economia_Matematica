Capítulo 2: Revisión de Aritmética y Álgebra

Descripción General

Este capítulo establece los fundamentos matemáticos esenciales para el análisis económico, cubriendo desde el sistema de números reales hasta las clasificaciones de funciones y sus aplicaciones económicas. Es un capítulo crucial que sienta las bases para todo el desarrollo posterior del curso.

Objetivos de Aprendizaje

Al finalizar este capítulo, el estudiante será capaz de:

  1. Comprender y operar con el sistema de números reales (\(\mathbb{R}\)), distinguiendo entre números racionales e irracionales
  2. Aplicar los principios de teoría de conjuntos para expresar relaciones matemáticas y económicas
  3. Construir y analizar productos cartesianos y pares ordenados en contextos económicos
  4. Distinguir entre relaciones y funciones, identificando las propiedades que definen una función
  5. Clasificar funciones según su tipo (constantes, lineales, cuadráticas, racionales, exponenciales, logarítmicas)
  6. Manipular exponentes y logaritmos aplicando las reglas algebraicas correspondientes
  7. Seleccionar el nivel de generalidad apropiado (numérico, paramétrico, general) para modelar situaciones económicas

Conceptos Clave

Concepto Símbolo/Notación Descripción
Números reales \(\mathbb{R}\) Unión de racionales e irracionales
Producto cartesiano \(X \times Y\) Conjunto de pares ordenados
Función \(f: X \to Y\) Relación unívoca
Dominio \(\text{Dom}(f)\) Conjunto de valores de entrada
Imagen \(\text{Im}(f)\) Conjunto de valores de salida
Función lineal \(y = a_0 + a_1x\) Pendiente \(a_1\), intercepto \(a_0\)
Hipérbola \(xy = a\) Elasticidad unitaria
Exponencial \(f(x) = e^x\) Crecimiento continuo
Logaritmo natural \(\ln x\) Inversa de \(e^x\)

Aplicaciones Económicas Destacadas

Funciones Lineales

  • Función de demanda: \(Q_d = a - bP\) (pendiente negativa)
  • Función de consumo keynesiana: \(C = C_0 + cY_d\) (propensión marginal al consumo)
  • Función de costo total: \(C = CF + cv \cdot Q\) (costos fijos y variables)

Funciones Cuadráticas

  • Maximización de beneficios: \(\pi(Q) = R(Q) - C(Q)\)
  • Funciones de costo con rendimientos variables

Funciones Racionales

  • Demanda con elasticidad unitaria: \(PQ = k\) (gasto total constante)
  • Costo fijo promedio: \(\text{CFP} = CF/Q\)

Funciones Exponenciales

  • Crecimiento económico continuo: \(Q(t) = Q_0e^{rt}\)
  • Descuento continuo: \(V_P = V_F e^{-rt}\)

Funciones Logarítmicas

  • Elasticidades: \(\epsilon = \frac{d\ln Q}{d\ln P}\)
  • Modelos log-lineales en econometría

Errores Comunes a Evitar

  1. Confundir relación con función: No toda relación es función
  2. Olvidar restricciones del dominio: Especialmente en funciones racionales y logarítmicas
  3. Aplicar incorrectamente las leyes de exponentes: Revisar cada regla cuidadosamente
  4. Ignorar restricciones cualitativas: En economía, las variables suelen ser no negativas
  5. Mezclar niveles de generalidad: Ser consistente en la formulación del modelo
